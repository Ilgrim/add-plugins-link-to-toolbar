A simple plugin to add a plugins link to the main toolbar for admins.

== Description ==

This is a simple plugin created to make a more convientent avenue of going to the plugins page when viewing your site while logged in. All it does is add a plugins link to the admin Appearance toolbar.

== Installation ==

1. Upload the folder `add-plugins-link-to-toolbar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
